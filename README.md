# Wechat_SqlManage
一个基于个人微信管理数据库的方案,即使不会SQL语法也能轻松操作数据库,基于chat2db
## 通过发送自然语言经过chatgpt 转化成 SQL 语言之后发送到 chat2db 执行增删改查操作
